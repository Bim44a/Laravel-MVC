# Laravel-MVC
Mempelajari konsep framework, arsitektur MVC (Model-View-Controller), struktur folder, routing, dan cara kerja dasar Laravel. Laravel adalah framework PHP yang paling populer saat ini. Dibuat oleh Taylor Otwell pada tahun 2011. MVC (Model-View-Controller) adalah design pattern untuk memisahkan logic, data, dan presentation.
